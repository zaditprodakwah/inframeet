### **1.1 Hack 1: GitHub Actions sebagai Runner Cron Job Gratis (Bypass Vercel Timeout)**

Next.js Serverless di Vercel memiliki batas waktu eksekusi (*timeout*) 10 detik pada versi gratis. Kita dilarang menjalankan fungsi sinkronisasi terjadwal yang berat di sana.

* **Taktik Guerrilla:** Gunakan **GitHub Actions** sebagai mesin pemicu (*runner*) gratisan. GitHub menyediakan 2.000 menit komputasi gratis per bulan untuk repositori privat.  
* **Eksekusi:** Buat berkas alur kerja (*workflow*) GitHub Actions yang berjalan setiap jam untuk mengetuk *endpoint* API rahasia Supabase, atau langsung memicu skrip sinkronisasi di *Local Worker* melalui koneksi aman SSH/Webhook.

### **1.2 Hack 2: Siluman Pengikis Data (Stealth Scraping) pada Local Headless Worker**

Jika skrip otomatis di MacBook Air Anda melakukan pengikisan (*scraping*) data secara agresif ke situs pemerintah (OSS/Kemenkumham) atau pangkalan riset (SINTA), alamat IP rumah Anda akan segera diblokir (*IP Banned*).

* **Taktik Guerrilla:** Gunakan teknik rotasi IP menggunakan **Mobile Tethering Interface** dengan skrip makro.  
* **Cara Kerja:** *Local Worker* dihubungkan ke jaringan internet ponsel Anda (*Tethering*). Skrip Node.js pada MacBook akan mendeteksi jika kecepatan respons melambat (terindikasi terkena *rate limit*), kemudian secara otomatis memicu perintah via USB ADB (*Android Debug Bridge*) atau Apple Script untuk mematikan dan menyalakan kembali *Airplane Mode* pada ponsel. Ini memaksa operator seluler memberikan alamat IP publik yang baru dalam waktu 3 detik. Alamat IP Anda selalu berubah secara dinamis secara gratis tanpa perlu menyewa *residential proxy* berbayar\!

### **1.3 Hack 3: Sistem Verifikasi DNS Instan Tanpa Query Port (The Dig Bypass)**

Untuk memverifikasi klaim kepemilikan domain (/dashboard/claim) melalui DNS TXT record, biasanya server membutuhkan kueri DNS yang memakan waktu.

* **Taktik Guerrilla:** Gunakan **Cloudflare Trace API** publik secara asinkron dari browser pengguna untuk mengonfirmasi DNS secara instan tanpa membebani server awan Anda.  
* **Endpoint Gratis:** https://cloudflare-dns.com/dns-query?name=\_inframeet-claim.domain.com\&type=TXT dengan header Accept: application/dns-json. Ini memindahkan seluruh beban resolusi DNS dari Supabase ke infrastruktur Cloudflare Edge secara cuma-cuma.

RETURNS TRIGGER AS $$  
BEGIN  
  \-- Inisiasi metadata default jika baru di-scrape  
  IF NEW.owner\_id IS NULL THEN  
    NEW.verified\_status := 'unverified';  
    NEW.trust\_score := 35; \-- Skor dasar keabsahan pendaftaran awal

### **4.2 Script Node.js: Siluman Rotasi IP di MacBook MBA (Local Worker Script)**

Skrip otomatisasi lokal untuk mendeteksi pemblokiran (*IP Banned*) dan melakukan rotasi alamat IP seluler secara gratis melalui USB debugging.

// scripts/guerrilla-ip-rotator.js (Dijalankan di MacBook Air)  
import { exec } from 'child\_process';  
import fetch from 'node-fetch';

async function checkIpAndRotate() {  
  try {  
    const res \= await fetch('\[https://api.ipify.org?format=json\](https://api.ipify.org?format=json)');  
    const { ip } \= await res.json();  
    console.log(\`\[+\] Alamat IP Publik Saat Ini: ${ip}\`);

    // Uji apakah terkena blokir oleh situs target (SINTA/OSS)  
    const testFetch \= await fetch('\[https://sinta.kemdikbud.go.id/\](https://sinta.kemdikbud.go.id/)', { timeout: 4000 });  
    if (testFetch.status \=== 429 || testFetch.status \=== 403\) {  
      console.warn("\[⚠️\] Terdeteksi Rate Limit/IP Banned\! Memulai rotasi seluler...");  
      await rotateMobileTethering();  
    } else {  
      console.log("\[✅\] Koneksi aman. Melanjutkan proses pengikisan data...");  
    }  
  } catch (err) {  
    console.error("\[⚠️\] Koneksi terputus/Timeout. Memulai rotasi seluler...");  
    await rotateMobileTethering();  
  }  
}

function rotateMobileTethering() {  
  return new Promise((resolve) \=\> {  
    // Jalankan perintah ADB via USB tethering ponsel Android untuk menyalakan/mematikan Airplane Mode  
    exec('adb shell settings put global airplane\_mode\_on 1 && adb shell am broadcast \-a android.intent.action.AIRPLANE\_MODE \--ez state true', () \=\> {  
      console.log("    \[-\>\] Mode Pesawat AKTIF (IP sedang dilepas...)");  
      setTimeout(() \=\> {  
        exec('adb shell settings put global airplane\_mode\_on 0 && adb shell am broadcast \-a android.intent.action.AIRPLANE\_MODE \--ez state false', () \=\> {  
          console.log("    \[-\>\] Mode Pesawat NONAKTIF (Mengikat IP baru...)");  
          setTimeout(() \=\> {  
            resolve();  
          }, 4000); // Tunggu sinyal seluler kembali stabil  
        });  
      }, 2000);  
    });  
  });  
}
