# INFRAMEET | Technical Documentation & Design Specification
## Project: Infrastructural Integrity Redesign (v4.2.0)
## Role: Principal Full-Stack Architect & Digital Marketing Engineer

---

### 1. Brand Identity & Psychological Framework
**Vision:** Transition from "Service Provider" to "Integrity Infrastructure."
**Psychological Anchors:**
- **Authority:** Reinforced through technical density and Monospace typography.
- **Evidence-Based Trust:** Every metric is anchored by a cryptographic hash (SHA-256).
- **Zero-Friction:** Modular "Bento Box" layouts for rapid information scanning.

### 2. Design System Architecture
**Core Palette (Deep Space Modernity):**
- `Primary (Cyber Purple)`: #6366f1 — Represents innovation, speed, and cloud-native architecture.
- `Secondary (Trust Blue)`: #3b82f6 — Represents academic rigor and reliability.
- `Surface-Base`: #0b0f10 — Deep Space Blue for professional depth.
- `Accent-Emerald`: #10b981 — Success signals and verified statuses.
- `Accent-Ruby`: #ef4444 — Critical alerts and integrity breaches.

**Typography:**
- **Narrative/UI:** `Plus Jakarta Sans`. A geometric sans-serif that balances modern tech with readability.
- **Data/Technical:** `JetBrains Mono`. High-legibility monospace for hashes, system IDs, SLA metrics, and logs.

### 3. Component Engineering
- **TopAppBar (Mega-Menu Enabled):** Splits the user journey into "Enterprise B2B" and "Academic Integrity" tracks.
- **Glassmorphism Panels:** Uses `backdrop-blur-xl` and subtle borders (`border-white/10`) to create depth without clutter.
- **Telemetry Cards:** Real-time data viz components using Framer Motion for numerical count-ups and fluid transitions.
- **The Formulator (ROI Calculator):** A multi-step wizard integrating Recharts for live projection of infrastructure savings.

### 4. Technical Stack Compliance
- **Frontend:** Next.js 16 (App Router), Tailwind CSS v4, Framer Motion, Recharts.
- **Backend/Logic:** TypeScript Strict Mode, Supabase (PostgreSQL), Xendit Invoicing integration.
- **Academic Protocols:** Turnitin No-Repository, SmartPLS 4, SPSS Statistics integration.
- **Security:** AES-256-GCM Encryption, SHA-256 Attestation, UU PDP (Indonesian Privacy Law) Compliance.

### 5. Deployment & Performance Metrics
- **Infrastructure:** Vercel Serverless + AWS Edge Nodes.
- **Performance Target:** Lighthouse Score > 95 (Core Web Vitals Optimized).
- **Availability:** 99.999% SLA backed by multi-cloud redundancy.
