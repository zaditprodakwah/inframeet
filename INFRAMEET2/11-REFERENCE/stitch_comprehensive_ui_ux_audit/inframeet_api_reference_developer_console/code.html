Authentication

INFRAMEET uses API keys and OAuth 2.0 to authenticate requests. All API requests must be made over HTTPS. Requests made over plain HTTP will fail.

/v1/deployments

Creates a new deployment infrastructure node in the specified region. This endpoint triggers a series of cryptographic checks before provisioning resources.

PARAMETERS

cluster_id
String • Required
The unique identifier of the target infrastructure cluster.
region
String • Optional
The geographic region code (e.g., us-east-1). Defaults to system-optimal.
NODE: US-WEST-4
REQUEST
curl -X POST https://api.inframeet.com/v1/deployments \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "cluster_id": "inf-9923-x",
    "region": "us-west-4"
  }'
