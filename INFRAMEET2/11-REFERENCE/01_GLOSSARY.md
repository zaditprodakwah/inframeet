# INFRAMEET - Definitive Glossary of Terms

A reference explaining all domain-specific terminology used across the INFRAMEET application.

- **Omni Storefront**: The SEO-optimized directory profile page created for an entity listing.
- **BAST (Berita Acara Serah Terima)**: Work completion certificate used as the baseline trigger for verified buyer status and escrow settlement.
- **Reputation Decay**: The automated daily loss of -5 reputation points when no verified activity is recorded for 60 consecutive days.
- **Trust Badge**: Embedded Web Widget displaying the dynamic, verified trust score of an entity on partner storefronts.
- **Staging Inbox**: A queue table (`staging_inbox`) designed to stage raw data inputs (e.g., RSS, spam logs) before ingestion.
- **Turnstile Captcha**: A zero-interaction captcha service provided by Cloudflare used to protect public forms.
