# INFRAMEET - Product Glossary

- **Trust Score**: A dynamically calculated number from 0 to 100 representing the verified credibility of an entity listing.
- **BAST (Berita Acara Serah Terima)**: Work completion certificate used as the baseline trigger for verified buyer status and escrow settlement.
- **Reputation Decay**: The automated daily loss of -5 reputation points when no verified activity is recorded for 60 consecutive days.
- **Staging Inbox**: A queue table (`staging_inbox`) designed to stage raw data inputs (e.g., RSS, spam logs) before ingestion.
